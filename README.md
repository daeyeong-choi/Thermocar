# 체온 측정 디바이스 설계

## 1. 프로젝트 개요

- 본 프로젝트는 병원 등 의료 환경에서 **아이들이 체온 측정에 두려움을 느끼는 문제**를 해결하고자 기획되었습니다.
- 아이들이 **놀이처럼 체온을 측정**할 수 있도록 유도하는 **헬스케어 디바이스 설계**를 목표로 합니다.

**진행 기간**: 2023년 9월 ~ 12월  
**참여자**: 최대영 

---

## 2. 기구 설계 컨셉

### 문제 인식

- 병원에서 체온을 재려는 순간 우는 아이를 관찰
- 익숙하지 않은 경험 자체가 공포로 다가옴
- '측정'이 아닌 '놀이'라는 인식을 부여하고자 함

### 목표

- **아이 주도적 체온 측정 경험 제공**
- **친근한 디자인** 채택
- **놀이 요소** 포함
